# Travel Planner

A 2-step travel planning SPA built with Vanilla JavaScript and Vite.

## Features

- Collect trip data (destination, dates, travelers, budget, interests, preferences)
- Generate AI-powered travel plans with weather, maps, flights, hotels, and activities
- Save trips to LocalStorage
- SPA routing with History API
- Observer/PubSub state management

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run preview` - Preview production build
- `npm run test` - Run unit tests
